   TDM 浏览器扩展 - 安装说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【Chrome / Edge / Brave】

1. 打开 chrome://extensions
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录（包含 manifest.json 的文件夹）
5. 安装完成后复制扩展 ID
6. 打开 TDM 主程序 → 浏览器扩展设置 → 填入扩展 ID
7. 重启 TDM 主程序

【Firefox】

1. 打开 about:debugging#/runtime/this-firefox
2. 点击「临时载入附加组件」
3. 选择本目录中的 manifest.json

【注册 Native Host】

如果浏览器提示「找不到 Native Messaging Host」，
请在管理员 PowerShell 中运行安装包同目录的：
   register_native_host.bat

  配合 TDM 主程序使用：
  • 浏览器中所有下载自动拦截
  • 图片/视频/音频旁显示 TDM 下载按钮
  • 右键「保存到 TDM」一键添加
